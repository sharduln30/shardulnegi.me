# shardulnegi.me

Personal portfolio website for Shardul Negi — Senior Software Engineer.

**Live:** [shardulnegi.me](https://shardulnegi.me)

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **UI:** React 19, TypeScript
- **Styling:** Tailwind CSS v4
- **Animation:** Framer Motion, GSAP + ScrollTrigger, Lenis (smooth scroll)
- **Fonts:** Space Grotesk, Inter, JetBrains Mono

## Sections

Hero · About · Skills · Experience · Journey Timeline · Projects · Achievements · Education · Blog · Resume · Contact

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Auto-deploys to Vercel on push to `main`.
